READ ME multipagewebsite
	
	##Site Description

	Music Exposure is a mock music review/blog/recommendation website created using CSS and HTML.
	This website is built to adapt to mobile, computer, and different window sizes.  
	This website features 3 pages.

	The "Home" page has a little welcome message and shows the "User Weekly Music Pick"
		This is currently displaying "DANCE..." by Slayyyter
			including: a image of the album cover and a YouTube embed for the song.
	
	The "Music" page is where "Our Weekly Music Pick" is displayed (chosen by the curator).
		This is currently displaying "For The Departed" by Shayfer James
			including: a image of a poster for Shayfer James' tour and a YouTube embed for the song.
	
	The "Contact Us" page is where user's can contact us for "Weekly User Pick", "Chat", or "Other"
		This form is mainly meant for users like "Callie" to submit songs for the Weekly User Pick (on the "Home" page)
			This mock form current leads to "action_page.php" which doesn't exist
			The form uses minlength and required to ensure the user fills out every field.
